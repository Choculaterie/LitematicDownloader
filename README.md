# Public Repo For The LitematicDownloader Mod

The main branch is always up to date with the latest version, check out the other branches if you want a specific outdated version!

https://modrinth.com/mod/litematicdownloader

Compile yourself in Intellij IDEA :

export JAVA_HOME=/usr/lib/jvm/java-21-openjdk  
export PATH=$JAVA_HOME/bin:$PATH

./gradlew clean build -x test
